CSS Grid Свойства

Контейнер

display: grid – создает grid-контейнер
grid-template-columns – задает колонки (1fr 2fr 1fr)
grid-template-rows – задает строки
gap – расстояние между элементами
grid-template-areas – именованные области
justify-items – выравнивание по горизонтали
align-items – выравнивание по вертикали

Элементы

grid-column – позиция по колонкам (1 / 3)
grid-row – позиция по строкам (span 2)
grid-area – имя области
justify-self – свое выравнивание по горизонтали
align-self – свое выравнивание по вертикали
