Flexbox Свойства

Контейнер

display: flex – создает flex-контейнер
flex-direction – направление (row, column)
justify-content – выравнивание по главной оси
align-items – выравнивание по поперечной оси
flex-wrap – перенос элементов (wrap)
align-content – выравнивание строк
gap – расстояние между элементами

Элементы

flex-grow – коэффициент растяжения
flex-shrink – коэффициент сжатия
flex-basis – базовый размер
flex – краткая запись (grow shrink basis)
align-self – свое выравнивание
order – порядок отображения
